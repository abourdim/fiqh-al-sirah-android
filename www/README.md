# 📜 فقه السيرة — Fiqh Al-Sirah

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual app based on "Understanding the Prophet's Biography" by Sheikh Mohammed Al-Ghazali.

## Features
- 20 Sirah cards covering the Prophet's life
- Trilingual: Arabic, English, French
- Narrator with karaoke mode
- Quiz: Who Wants to Be a Scholar?
- XP & badges system
- 3 themes, PWA support
